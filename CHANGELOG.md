# Changelog

All notable changes to the BIS Alazhar Mobile Application will be documented in this file.

## [1.0.0] - 2026-01-30

### Added
- Initial release of BIS Alazhar Mobile Application
- User authentication (Login & Registration)
- Home dashboard with service tiles
- Class schedule viewer
- Study materials browser
- Course notes ordering system
- Student forum with posts and comments
- User profile management
- Push notifications support
- Secure token-based authentication
- Dynamic background UI components
- Responsive Material Design 3 interface

### Features
- **Authentication**
  - University ID based login
  - Secure password handling
  - Persistent sessions
  - User registration with avatar selection

- **Academic Services**
  - View personalized class schedules
  - Browse subjects by year and term
  - Access study materials and files
  - External file viewer integration

- **Notes System**
  - Browse available course notes
  - View note details and pricing
  - Place note orders

- **Forum**
  - Create and view posts
  - Comment on discussions
  - Upvote system
  - Real-time updates

- **Profile**
  - View personal information
  - Edit profile (coming soon)
  - Logout functionality

- **Notifications**
  - View official announcements
  - Mark notifications as read
  - Real-time notification badges

### Technical
- Flutter 3.0+ support
- Dio HTTP client integration
- Secure storage implementation
- RESTful API integration
- Material Design 3
- Feature-based architecture

### Known Issues
- Backend API URL needs configuration
- Some features require backend implementation
- Dark mode not yet implemented
